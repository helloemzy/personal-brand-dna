Open Graph Images Required
==========================

The following Open Graph preview images need to be created and placed in the public folder:

1. og-image.jpg (1200x630px)
   - Default social sharing image for the homepage
   - Should showcase the BrandPillar AI brand and value proposition
   - Include logo, tagline, and visual elements

2. og-results.jpg (1200x630px)
   - Preview image for workshop results pages
   - Should show a sample Brand House or archetype visual
   - Include "Discover Your Brand DNA" messaging

3. og-share.jpg (1200x630px)
   - Preview image for shared results pages
   - Should be generic enough for any archetype
   - Include BrandPillar branding and CTA

4. og-article.jpg (1200x630px)
   - Preview image for content/article pages
   - Should represent thought leadership and content
   - Include BrandPillar logo

Design Guidelines:
- Use brand colors (blue-600 to blue-800 gradient)
- Include BrandPillar AI logo
- Ensure text is readable at small sizes
- Use high-quality images
- Optimize file size (under 300KB each)

Tools for Creation:
- Canva (templates available)
- Figma
- Adobe Creative Suite
- Bannerbear API (for dynamic generation)
